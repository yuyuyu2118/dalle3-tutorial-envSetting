# クイックスタート

## 1. リポジトリをクローンする

```bash
git clone
```

## 2. ディレクトリに移動する

```bash
cd
```

## 3. nodeを初期化する

```bash
npm init -y
```

## 4. パッケージをインストールする

```bash
npm install express openai cors
```

## 5. package.jsonのスクリプトを修正する

```json
"scripts": {
    "start": "node server.js"
  },
